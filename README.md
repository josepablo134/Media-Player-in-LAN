# Media-Player-in-LAN
This is a Media Player based on LAN server using python, java and php, running on raspberry pi.

# Description
This system shows an php page in the LAN that allows to the user configure the system and songlist,
and control the play of the music.
To start, is necesary initialize the services using the configuration page. The services is a group
of python scripts that controls the playlist and playing.
This system have a lot of glitches and it is improvable. Once started all the services, the system fulfills its mission.
This system will be fixed soon.
This code is open for its improvement.
